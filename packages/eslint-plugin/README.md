# `@mozisan/eslint-plugin`

## Installation

```sh
npm i -D @mozisan/eslint-plugin
```

## Usage

```json
{
    "extends": ["plugin:@mozisan/preset"]
}
```

```json
{
    "extends": ["plugin:@mozisan/preset-react"]
}
```

```json
{
    "extends": ["plugin:@mozisan/preset-next"]
}
```

```json
{
    "extends": ["plugin:@mozisan/preset-tailwindcss"]
}
```
